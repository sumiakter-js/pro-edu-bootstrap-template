Flex Box CSS. The Flexible Box Module, usually referred to as flexbox, was designed as a one-dimensional layout model, and as a method that could offer space distribution between items in an interface and powerful alignment capabilities.

A semantic element clearly describes its meaning to both the browser and the developer. Examples of semantic elements:
form tag, table tag, and article tag - Clearly defines its content. Examples of non-semantic elements: div and span - Tells nothing about its content.

The one-dimensional Flexbox: CSS Flexible Box Layout (or Flexbox) allows designers to position responsive elements appropriately within screens of different sizes. The two-dimensional Grid: CSS Grid aligns items in columns and rows, allowing developers to easily control the rendering and appearance of large layouts and whole pages meant for the desktop, tablet, and smartphone displays.

Tailwind: Tailwind CSS uses a set of utility classes to create a neat UI with more flexibility and uniqueness.
Bootstrap: Sites created using Bootstrap follow the generic pattern that makes them look identical.
